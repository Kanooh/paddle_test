Gebruik de kracht van Google Custom Search om de zoekfunctie van je website uit te breiden. Met Custom Search help je bezoekers inhoud binnen en buiten je website te vinden.

De Google Custom Search is een uitbreiding van de standaard zoekfunctie. Deze paddlet maakt het mogelijk ook relevante resultaten van andere websites terug te vinden. Je kan zelf het zoekbereik instellen op je Google Custom Search account.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’.
  • Vervolgens verschijnt de configuratiepagina.
  • Om de Google Custom Search te activeren, heb je twee Google instellingen nodig.
    • De Custom Search die je kan aanvragen op www.google.com/cse
    • De Google API key die je kan verkrijgen op console.developers.google.com
  • Om de paddlet zichtbaar te maken voor de websitebezoekers ga je naar ‘Theme Store’ > ‘Bewerk theme’ > ‘Header’. Bij de titel ‘Zoeken’ kan je de Google Custom Search activeren.
