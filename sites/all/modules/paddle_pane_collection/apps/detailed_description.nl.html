Voor zowel paginatypen als overzichtpagina's kan je pane collections definiëren en hergebruiken. Dit scheelt je veel tijd wanneer je meerdere pagina's aan gaat maken waarin dezelfde panes gebruikt worden. Deze functionaliteit gaat nog een stapje verder dan de globale regiosettings die per paginatype gelden.

Installatie

  • Klik op ‘Installeren’ om de paddlet te activeren
  • Maak een pane collection aan
  • Maak een pagina aan
  • Voeg een pane toe van het type 'Pane collection'
  • Kies je gewenste collection
  • Alle panes van die collection worden ineens op de pagina geplaatst
