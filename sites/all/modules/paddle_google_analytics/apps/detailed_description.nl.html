Krijg inzicht in het surfgedrag van jouw websitebezoekers op basis van de krachtige Google Analytics statistieken. Een ideale basis om data te verzamelen en je website verder te gaan optimaliseren.

Uit Google Analytics kan je veel leren over het surfgedrag van websitebezoekers. Je ziet niet alleen welke pagina’s worden aangeklikt, maar vindt ook demografische gegevens en interesses terug. Hiermee kan je aan de slag de inhoud en structuur van de website te optimaliseren.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’.
  • Vervolgens verschijnt de configuratiepagina.
  • Vul jouw Google Analytics-ID in, een unieke code voor elke website die je individueel wilt tracken. De samenstelling van een ID is als volgt: UA-XXXXXX-YY.
  • Je kan deze code verkrijgen op de website van Google Analytics, waar je nadien ook de statistieken kan bekijken.
