Toon widgets van andere websites op jouw website. Configureer de code eenmalig en hergebruik ze op eender welke pagina.

De Embed functie brengt veel dynamiek in je website. Je kan inhoud en widgets van externe websites, zoals een YouTube video, een weer- of kalenderwidget en Twitterfeed op pagina’s plaatsen. Door de inhoud te embedden hoeft de bezoeker jouw website niet te verlaten om deze content te consumeren.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’.
  • Ga vervolgens naar ‘Instellen’.
  • Klik op ‘Nieuwe widget’ en vul de titel en embed code in. De code vind je terug op de externe bronpagina. Soms moet je deze aanvragen bij de webmaster van de website.
  • Kies een bestaande pagina of maak een nieuwe aan.
  • Ga in de back-end naar ‘Pagina regio’s’ en maak een ‘Embed widget’ aan.
  • Je kan ook het kleurenpalet van een widget aanpassen.
  • Aanpassingen aan de widget in de Paddle Store worden automatisch op de pagina toegepast.
