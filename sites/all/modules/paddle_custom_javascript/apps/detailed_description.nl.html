Maak code beschikbaar op elke pagina van je website.

Door deze paddlet te activeren krijg je als sitebeheerder de mogelijkheid om een script in te geven. Dit script wordt op elke pagina geplaatst in de <head>.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’ om de 'Custom JavaScript' paddlet te activeren.
  • Geef het script in.
  • Het script is nu op elke pagina beschikbaar.
