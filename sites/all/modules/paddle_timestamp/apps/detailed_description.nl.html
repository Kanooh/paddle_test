Deze paddlet geeft je de mogelijkheid om een datum onder elke pagina te plaatsen (per inhoudstype), zodat je bezoekers zien wanneer de betreffende pagina voor het laatst bijgewerkt werd.

Installatie

  • Installeer de "Datum op pagina" paddlet
  • Geef aan op welke inhoudstypen je de datum wilt tonen
  • Je vind een nieuw veld op de editeerpagina met datum wanneer de pagina voor het laatst is bijgewerkt. Deze datum kan handmatig aangepast worden.
