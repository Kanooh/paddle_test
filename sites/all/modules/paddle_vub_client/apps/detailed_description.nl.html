Connecteer met de centrale VUB server om nieuwsberichten en events te delen en ontvangen.

Door deze paddlet te activeren kan je een connectie maken met de VUB server om nieuwsberichten die je gepubliceerd hebt te delen met andere VUB websites. Ook kan je op basis van een set van tags nieuwsberichten van andere VUB websites ophalen vanuit de VUB server.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’ om de 'VUB client' module te activeren.
  • Ga naar de configuratiepagina van de module en vul het VUB server adres in. Ook de tags van de nieuwsberichten die je wilt ophalen kan je hierin invullen.
  • Publiceer je nieuwsberichten om deze door te sturen naar de VUB webserver.
