Vraag een lijst met pagina titels en links op om ze te tonen op een andere website of in een mobiele app. Filter de lijst desgewenst op één of meerdere gelinkte trefwoorden.

Een webservice laat verschillende toepassingen toe om je website te bevragen. Het biedt een Application Programming Interface, ook wel gekend als API. We bieden een REST webservice die antwoordt in JSON formaat.

Gebruik

  1. Installeer de webservice paddlet.
  2. Surf naar jouw domeinnaam plus 'api'. Bijvoorbeeld http://example.com/api
  3. Je ontdekt welke onderdelen van je website bevraagd kunnen worden.
  4. Kies bijvoorbeeld pagina's door naar http://example.com/api/v1.0/pages te surfen.
  5. Je krijgt een lijst van gepubliceerde pagina's.
  6. Verfijn je bevraging door te filteren, sorteren en/of pagineren. Surf bijvoorbeeld naar http://example.com/api/v1.0/pages?filter[general-vocabulary.term]=water&sort=title&range=5&page=2 om pagina 6 tot en met 10 opgelijst te krijgen van de lijst gepubliceerde pagina's gesorteerd op pagina titel, gelinkt aan het trefwoord 'water'.
