Integreer de MailChimp diensten in je website en maak nieuwsbrieven in een handomdraai. Het is met deze paddlet ook mogelijk een inschrijfformulier in een pane te plaatsen.

Met een nieuwsbrief bereik je bezoekers actief en breng je hen op de hoogte van nieuwigheden, belangrijke informatie en weetjes. Met de MailChimp paddlet kan je zowel inschrijfformulieren plaatsen om een publiek op te bouwen als nieuwsbrieven verzenden. Vul je brief op met bestaande inhoud of schrijf nieuwe teksten.

Installatie en configuratie

  • Klik bovenaan op ‘Installeren’.
  • Om van start te gaan moet je kañooh website eerst toegang krijgen tot je MailChimp profiel. Dit doe je door de juiste API Key in te vullen. Deze kan je na aanmaak van je MailChimp account terugvinden via: http://admin.mailchimp.com/account/api. Of klik in MailChimp rechts bovenaan op ‘Profile’ > ‘Extras’ > ‘API Keys’.
  • Ga naar de configuratiepagina van de Paddlet en vul de API Keys aan.
  • Klik op ‘+ Inschrijfformulier’ om een formulier aan te maken. Deze kan je vervolgens als pane in een regio plaatsen.
  • Een nieuwsbrief opstellen doe je via ‘Inhoud’ > ‘Inhoud toevoegen’ > ‘Nieuwsbrief’.
